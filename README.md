# 🌍 Air Quality Analysis & AQI Prediction

## 📌 Project Overview
This project focuses on analyzing air pollution data across different states and cities in India. The goal is to understand pollution patterns and build a machine learning model to predict pollution levels.

---

## 📊 Dataset Description
The dataset contains:
- State and city information  
- Pollutant types (PM2.5, PM10, NO2, CO, etc.)  
- Minimum, maximum, and average pollution values  
- Date-time information  

---

## 🎯 Objectives
1. Analyze average pollution levels for different pollutants  
2. Compare pollution levels across states  
3. Identify top polluted cities  
4. Study distribution of pollution values  
5. Analyze relationships between pollution metrics  
6. Visualize correlations using heatmaps  
7. Build a Linear Regression model for prediction  

---

## 🛠️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

---

## 📊 Exploratory Data Analysis (EDA)
Performed EDA using:
- Bar charts (pollutant & state analysis)  
- Histogram (distribution analysis)  
- Scatter plots (relationship analysis)  
- Heatmap (correlation analysis)  
- Boxplot (outlier detection)  

---

## 🤖 Machine Learning
- Model: Linear Regression  
- Target: Pollution Average (PM2.5)  
- Evaluation Metrics:
  - R² Score  
  - Mean Absolute Error (MAE)  
  - Mean Squared Error (MSE)  
  - Root Mean Squared Error (RMSE)  

---

## 📈 Key Insights
- PM10 and PM2.5 are major contributors to pollution  
- Some states show consistently high pollution levels  
- Certain cities act as pollution hotspots  
- Data contains outliers indicating extreme pollution  
- Strong correlation exists between pollution metrics  

---

## 🧠 Conclusion
Air pollution is a serious concern in many regions. Data analysis and machine learning can help in understanding patterns and predicting pollution levels, enabling better decision-making.
